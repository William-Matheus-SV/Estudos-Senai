# Sistema de Inscrição SENAI
# https://william-matheus-sv.github.io/Atividade-31-10-2025/

Sistema web para inscrição e enriquecimento de dados de alunos do SENAI.

## Funcionalidades

- **Formulário de Inscrição**: Cadastro inicial de novos alunos
- **Formulário de Enriquecimento**: Completação de dados para alunos já cadastrados
- **Validação de campos**: CPF, telefone, CEP formatados automaticamente
- **Design responsivo**: Funciona em desktop e mobile

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript Vanilla

## Como usar

1. Abra o `index.html` em um navegador
2. Para novos alunos: Preencha o formulário de inscrição
3. Para alunos cadastrados: Clique em "Já sou aluno" para acessar o enriquecimento de dados

## Estrutura do Projeto

- `index.html` - Página inicial com formulário de inscrição
- `index.css` - Estilos da página inicial
- `script.js` - JavaScript da página inicial
- `enriquecimento.html` - Formulário de enriquecimento de dados
- `enriquecimento.css` - Estilos do formulário de enriquecimento
- `enriquecimento.js` - JavaScript do formulário de enriquecimento
